# Navigate the app

Use for "how do I…" / "where is…" questions about the product.

## How to answer
1. Read `knowledge/app-overview.md` for pages, statuses, gates, and integrations.
2. Give a short, step-by-step answer naming the exact page and control.

## Quick map
- Approve a **revenue plan** → open the contract (`/customers/[id]`) → **Revenue Plan** tab → Gate 1 Approve.
- Approve / post **journal entries** → same contract → **Bills / Journal Entries** → Gate 2 (controller).
- Add a contract → **Customers** (`/`) → Add; or it auto-ingests from Salesforce.
- See what needs sign-off → **Dashboard** → "Needs your attention".
- Inspect a contract's steps → contract page → progress timeline + **Audit** tab.

## Remember
You explain how to act; the human performs irreversible actions (approve, post, send).
